# 陰陽論

## 概念
漢方医学の根本思想。あらゆる事象を「陰」と「陽」の対立・調和で捉える。

## 陰陽の性質

| 陰 | 陽 |
|----|----|
| 寒・冷 | 熱・温 |
| 静・下降 | 動・上昇 |
| 裏・内 | 表・外 |
| 虚・弱 | 実・強 |
| 夜・暗 | 昼・明 |

## 漢方診療での応用
- **陰証**：体が冷え、機能が低下した状態（例：附子剤が適応）
- **陽証**：熱・炎症・亢進した状態（例：石膏・黄連剤が適応）

## 臨床的ポイント
- 陰陽は固定でなく相互転化する（陽が極まれば陰に転じる）
- 薬剤師として重要：熱証に温熱薬を用いると悪化するため、証の見極めが必須
